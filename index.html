This page contains information about the new scrypt based BBQcoin (BQC).

What is BBQcoin?

BBQCoin is a fast version of Litecoin.

Features:

  • 60 seconds block targets
  • Total of 88 million coins
  • 42 coins per block
  • 60 blocks to retarget difficulty
  • Scrypt hashing algorithm
